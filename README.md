websocket-demo
==============================
-  Working environment:
  * $> source myenv/bin/activate
-  Tornado:
  * $> sudo apt-get install python-tornado 
-  JSON:
  * $> sudo apt-get install python-simplejson

- PiFace server:
  * $> git clone https://github.com/tonychan217/websocket-demo.git
  * $> cd websocket-demo
  * $> python server.py

- Browser: 
  * http//[your_raspberrypi_IP]:8888
